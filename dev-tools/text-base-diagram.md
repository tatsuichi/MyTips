# テキストベースの作図

作図ツール

## [PlanUML](https://plantuml.com/ja/) 

### 実行環境

- WSL（Ubuntu）

### インストール手順 

以下の VS Code の拡張機能をインストールする

- [PlantUML](https://marketplace.visualstudio.com/items?itemName=jebbs.plantuml)

``` sh
# Javaのインストール
sudo apt install -y default-jdk

# 状態遷移図を使用する場合は、GraphVizのインストールが必要
sudo apt install graphviz
```
